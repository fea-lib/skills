# Chart development

Read `references/chart-conventions.md` before starting any chart work. It contains all hard constraints, layout recipes, theming patterns, component-specific design notes, anti-patterns, and the pre-ship checklist.

To choose the right chart type, read `references/chart-decision-tree.md` — a compact lookup table covering all six data families.
